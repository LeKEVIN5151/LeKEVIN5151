## 📡 Telecommunications Engineering — Academic Projects

### 🔹 Private Mobile Network Deployment via SDR
*End-to-end implementation of GSM/LTE networks in a lab environment.*
- **Hardware:** Ettus USRP B210 for full-duplex RF processing
- **Stack:** srsRAN (4G/LTE eNodeB + EPC) and Osmocom (2G/GSM)
- **Focus:** Comparative analysis of access methods and protocols between GSM and LTE — differences, advantages and performance tradeoffs via Wireshark

### 🔹 High-Capacity Optical Network — REFEFO
*Design of an optical link based on Arsat's REFEFO tender (Tramo I).*
- Selected equipment and optical links (fiber, amplifiers, ROADM, FOADM)
- Analyzed chromatic dispersion, power budget, OSNR and nonlinear effects
- Guaranteed redundancy and scalability across the designed topology

### 🔹 GPON Network with Full Observability Stack
*Design and prototype of a GPON network for Internet, IPTV, OTT and VoD services.*
- Configured VPN tunnels and MikroTik routers with Ministra for IPTV management
- Deployed observability stack: Prometheus, Grafana, Loki and SNMP Exporter on Docker
- Used VXLAN for network segmentation; validated all services in lab environment
